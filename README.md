# Mandelbrot set

Simple mandelbrot set demo written in C++ with SDL2 library.

![Preview Image 1](preview1.jpg)
![Preview Image 2](preview2.jpg)
![Preview Image 3](preview3.jpg)

## Compile
`
make
`

## Run
`
make run
`

## Clean
`
make clean
`

## Controls
| Key        | Description |
| ---------- | ----------- |
| W,A,S,D    | movement    |
| Left Shift | speed up    |
| Left Ctrl  | slow down   |
| E or +     | zoom in     |
| Q or -     | zoom out    |
