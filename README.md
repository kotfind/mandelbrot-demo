# Mandelbrot set
## Compilation
`
make
`

## Running program
`
make run
`

## Controls
| Key        | Description |
| ---------- | ----------- |
| W,A,S,D    | movement    |
| Left Shift | speed up    |
| Left Ctrl  | slow down   |
| E or +     | zoom in     |
| Q or -     | zoom out    |

## Cleaning
`
make clean
`
